----------------------------------------------------------------
SAVOL 1.
Template (shablon) nima va Django'da u nima uchun kerak?
O'z so'zlaringiz bilan tushuntiring.

JAVOB:
Django'da u ma'lumotlarni ekranga chiroyli chiqarish uchun ishlatiladi.


----------------------------------------------------------------
SAVOL 2.
render() funksiyasi nima qiladi? U qanday 3 ta ishni bajaradi?

JAVOB:
1 .Template (HTML)ni topadi.
2 .Ma'lumotlarni templatega uzatadi.
3 .Tayyor sahifani foydalanuvchiga yuboradi.


----------------------------------------------------------------
SAVOL 3.
Context nima? View'dan template'ga ma'lumot qanday uzatiladi?

JAVOB:
Ma'lumot lug'at (dict) ko'rinishida render() funksiyasi orqali uzatiladi.


----------------------------------------------------------------
SAVOL 4.
{{ }} va {% %} o'rtasidagi farq nima? Har biriga misol keltiring.

JAVOB:
{{ }} — o‘zgaruvchi (ma'lumot)ni chiqaradi.
Misol: {{ ism }}
{% %} — buyruq va mantiqiy amallar uchun ishlatiladi.
Misol: {% for kitob in kitoblar %}


----------------------------------------------------------------
SAVOL 5.
Nima uchun template fayllarni books/templates/books/ ko'rinishida
(ikki marta app nomi bilan) joylashtiramiz?

JAVOB:
Bu usulda Django qaysi app'ning template'i ekanini aniq biladi. Bir nechta app'da index.html bo'lsa ham, chalkashmaydi.

----------------------------------------------------------------
SAVOL 6.
{% extends %} va {% include %} o'rtasidagi farq nima?
Har biri qachon ishlatiladi?

JAVOB:
extends — asosiy shablondan foydalanadi.
include — kichik qismlarni (masalan, header, footer) qo'shadi.
----------------------------------------------------------------
SAVOL 7.
Quyidagi kod template'da nima qaytaradi va nima uchun?

    {{ kitob.muallif.ism_familiya }}

JAVOB:

Sababi: kitob → muallif obyektiga o'tadi, so'ng ism_familiya maydonining qiymatini ekranga chiqaradi.

----------------------------------------------------------------
SAVOL 8.
Filter nima? Quyidagi filtrlar nima qiladi:
  |upper, |length, |default, |date

JAVOB:
|upper — matnni katta harflarga o'tkazadi.
|length — uzunligini (elementlar sonini) qaytaradi.
|default — qiymat bo'lmasa, standart qiymat chiqaradi.
|date — sanani kerakli formatda chiqaradi.
